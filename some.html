Opcode Description Format
MOVE Move a value from one register to another R(A) = R(B)
LOADK Load constant R(A) = K(B)
LOADBOOL Load a boolean value R(A) = (B != 0); pc += C
LOADNIL Load nil into registers R(A) = NULL; R(A+1) = NULL; ...
GETUPVAL Get a value from an upvalue R(A) = UpValue[B]
SETUPVAL Set a value to an upvalue UpValue[B] = R(A)
GETTABUP Get a value from an upvalue table R(A) = UpValue[B][C]
SETTABUP Set a value to an upvalue table UpValue[B][C] = R(A)
GETTABLE Get a value from a table R(A) = R(B)[C]
SETTABLE Set a value to a table R(A)[B] = R(C)
NEWTABLE Create a new table R(A) = {} (size = B, C)
SELF Set a table field and return its value R(A+1) = R(B); R(A) = R(B)[C]
ADD Perform an addition R(A) = R(B) + R(C)
SUB Perform a subtraction R(A) = R(B) - R(C)
MUL Perform multiplication R(A) = R(B) * R(C)
DIV Perform division R(A) = R(B) / R(C)
MOD Perform modulus R(A) = R(B) % R(C)
POW Perform exponentiation R(A) = R(B) ^ R(C)
UNM Unary minus R(A) = -R(B)
NOT Logical NOT R(A) = not R(B)
LENGTH Get the length of a value R(A) = length of R(B)
CONCAT Concatenate multiple values R(A) = R(B) .. R(C) .. ...
JMP Jump to another instruction pc += sBx
EQ Compare two values for equality if R(B) == R(C) then pc += sBx
LT Compare two values for less-than if R(B) < R(C) then pc += sBx
LE Compare two values for less-than or equal if R(B) <= R(C) then pc += sBx
CALL Compare two values for less-than or equal if R(B) <= R(C) then pc += sBx
